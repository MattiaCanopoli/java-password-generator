# JAVA - PASSWORD GENERATOR

A simple password generator.

ask for infos to users with system.out and gather them with scanner(system.in)

### infos:
   1. name
   2. last name
   3. favorite color
   4. birth year
   5. birth month
   6. birth day
 
sum year + month + day

print a string with concatenation of name, last name, color and sum
